guess the word game

Project Brief:

A simple guess the word game that I built during my coursework at Skillcrush bootcamp. In this game, there is a list of words present, out of which our interpreter will choose 1 random word. The user will be asked to guess any alphabet. If the random word contains that alphabet, it will be shown as the output(with correct placement) else the program will ask you to guess another alphabet. User will be given 12 turns(can be changed accordingly) to guess the complete word.
